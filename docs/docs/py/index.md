## Python简介

Python是一种解释型高级编程语言 这意味着Python和PHP一样 都是解释性语言 Python的语法简洁且非常贴近自然语言(也就是我们说话的语言) 入门门槛极低<br>
- 特点: 写完直接运行 代码少 可读性强
- 运行: 只要有Python运行环境 就能运行代码 也可以使用pyinstall打包成.exe可执行文件在Windows上面运行

*缺点就是运行慢 不过Python仍然是最佳的编程入门语言*

## 为什么要学习Python

- 上手简单: 如果你想进入编程世界 我相信 很多人都会推荐你先学Python 因为的语法非常简单易懂 而且如果报错了 它会告诉你哪一行 哪个地方错了 这样非常方便改
- 用途广泛: 有很多的大型网站的后端用的就是Python 且Python也可以进行爬虫 方便从网络上快速获取知识 也可以用于自动化脚本等
- 海量知识: Python有着强大的社区 海量的第三方库 几乎就是开箱即用 也有pip这个Python自带的安装方式
