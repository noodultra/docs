# IDE介绍

*最后编辑于2026/06/07 by:nood*

IDE(Integrated Development Environment) 中文:集成开发环境 我们一般都叫IDE 是一种把程序员开发所需的各类工具整合到一起的的应用

核心组件:
- 代码编辑器: 提供语法高亮 自动补全等功能
- 编译器/解释器: 将源代码转成计算机可以执行的指令

## 常见IDE
- Visual Studio Code: 简称vscode 微软开发的代码编辑器 非常轻量的软件 写什么语言都可以 丰富的插件 如果你不止写一种编程语言 非常建议你用 但是配置比较麻烦 不过这些都不是问题! 面缶酱会教你的喵~<br>
*应该有人知道Visual Studio Code本质上不是IDE而是代码编辑器 不过因为Visual Studio Code确实好用 所有上面还是列出来了 不过还是要记住Visual Studio Code不是IDE 而是代码编辑器*
- IDLE: Python自带的IDE 全英文 如果你的电脑配置没那么好 可以试试这个
- PyCharm: JetBrains开发的IDE 写Python最好用的IDE
- Dev C++: 一个非常基础的专门为C/C++设计的IDE 如果你的电脑配置没那么好 可以试试这个
- Visual Studio: 简称vs 微软开发的一个IDE 写C系列语言 .NET首选
- IntelliJ IDEA: JetBrains开发的IDE 写Java Kotlin首选IDE
